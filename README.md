# Referral Management and Specialist Coordination Hub

A healthcare referral management platform designed to reduce referral leakage by helping providers route patients to appropriate in-network specialists, track referral progress, manage appointments, and improve care coordination.

## Tech Stack

- Frontend: Vue.js
- Backend: ASP.NET Core Web API
- ORM: Entity Framework Core
- Database: SQL Server

## Features

- Referral Intake & Tracking
- Specialist Assignment
- Appointment Scheduling
- Patient Dashboard
- Role-Based Access Control (RBAC)
- Referral Analytics
- Audit Logging

## Project Structure

```text
backend/
frontend/
database/
```

## Getting Started

### Backend

```bash
cd backend
dotnet restore
dotnet run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Database

Run the SQL scripts inside the `database` folder to create and seed the database.
